#rajula
Developed an interactive Sales Dashboard using Power BI to analyze and visualize sales performance data. The dashboard provides key insights into total revenue, sales by region, product performance, and customer trends. Implemented data cleaning, transformation, and visualization techniques to identify growth opportunities and improve decision-mak
The dataset was obtained from Kaggle and refined in Excel before importing into Power BI. It contains 5,000 transaction records from the years 2023 and 2024.
Key Fields:
Field Name
Description
Transaction ID
Unique ID for each transaction
Transaction Date
Date of payment (2023–2024)
Product Amount
Total amount of the transaction
Product Category
Type of product/service
Payment Method
UPI, Credit Card, Debit Card, Wallet, etc.
Location
Urban, Suburban, or Rural area
Merchant Name
Vendor or platform where payment was made
Transaction Fees
Fee charged per transaction
Transaction Status
 Successful, Pending, or Failed
